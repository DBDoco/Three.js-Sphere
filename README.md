## How To Use

Clone this repository
```bash
$ git clone https://github.com/DBDoco/Flappy-Bird-Clone.git
```
Go into the repository
```bash
$ cd Three.js-Sphere
```
Run development build
```bash
$ npm run dev
```

or

Build the project
```bash
$ npm run build
```
